5 comandos de Git que no conociamos

1-Git Diff

Muestra los cambios realizados a un archivo
Show changes made to a file

2-Git Tag

Muestra una lista de los Tag
Show a list of all tags

3-git add docs/*.txt

Añadimos todos los archivos dentro de un directorio y de una extensión especifica
We add all the files inside a directory and with a specific extension

4-git log --oneline --graph

Muestra graficos de los commits
Show graphs of commits

5-git clone URL/name.git NombreProyecto

Clona un proyecto de git en la carpeta NombreProyecto.
Clone a git project in the ProjectName folder.
